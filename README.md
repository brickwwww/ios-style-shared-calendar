# iOS 风格多用户共享日历

一个现代化的多用户共享日历应用，采用iOS风格设计，支持事项分类、隐私设置和用户协作。

## ✨ 功能特性

- 🎨 **iOS风格设计** - 采用苹果设计语言，毛玻璃效果
- 👥 **多用户支持** - 用户登录、访客模式
- 🏷️ **事项分类** - 工作、生活、学习、健康、社交、其他6种分类
- 🔐 **隐私保护** - 支持私人事项，仅创建者可见
- 🎯 **彩色标识** - 不同类型事项用不同颜色区分
- 📱 **响应式设计** - 适配移动端和桌面端

## 🚀 快速开始

### 在线使用
直接访问GitHub Pages链接即可使用（部署后会提供）

### 本地运行
```bash
# 克隆项目
git clone [您的仓库地址]
cd [项目名称]

# 使用Python启动服务器
python -m http.server 8080

# 或使用Node.js
npx serve -p 8080

# 访问 http://localhost:8080
```

## 📖 使用指南

1. **登录** - 输入用户名登录或选择访客模式
2. **浏览日历** - 点击日期查看当天事项
3. **添加事项** - 点击"+"按钮添加新事项
4. **选择分类** - 为事项选择合适的分类类型
5. **隐私设置** - 勾选"仅我可见"创建私人事项
6. **多人协作** - 与其他用户共享公开日程

## 🎨 事项分类

| 分类 | 颜色 | 说明 |
|------|------|------|
| 🏢 工作 | 红色 | 工作相关事项 |
| 🏠 生活 | 蓝色 | 日常生活事项 |
| 📚 学习 | 绿色 | 学习培训事项 |
| 💪 健康 | 橙色 | 健康运动事项 |
| 👥 社交 | 紫色 | 社交聚会事项 |
| 📝 其他 | 灰色 | 其他类型事项 |

## 💾 数据存储

- 使用浏览器本地存储（localStorage）
- 数据仅存储在用户设备上
- 支持多用户数据隔离和共享

## 🔧 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **样式**: CSS Grid, Flexbox, CSS Variables
- **存储**: LocalStorage API
- **部署**: GitHub Pages
